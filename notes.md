# AI Presentation Brief (10 + 5)

## Purpose

This presentation serves as an introduction to a series of presentations on AI for software developers. The goal is to establish shared context on AI evolution and current state, then use Q&A and follow-up sessions to identify which specific AI topics the audience wants to explore further. Subsequent presentations will be tailored based on audience interests and questions.

## Audience

- Software developers

## Goal

- Explain the evolution of AI with a focus on the biggest shifts.
- Define essential concepts as they appear (for example: machine learning, neural networks, generative AI).

## Constraints

- Presentation time: 10 minutes.
- Q&A time: 5 minutes.
- Keep content high-level and only cover major changes.

## Keep/Skip Rule

- Keep: major turning points, clear definitions, and real-world impact.
- Skip: minor milestones, technical deep dives, and long historical lists.

## Possible Future Presentations

### Technical Topics

- Attention mechanisms and Transformers
- Neural networks
- Vector embeddings and semantic search

### Practical Topics

- Prompt engineering techniques
- Using AI agent skills and agentic workflows
- Integrating AI tools into development workflow (Copilot, code generation)
- Evaluating and testing AI-generated code
